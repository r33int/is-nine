# is-nine
NPM package for checking if a number is equal to 9.

## Installation

``` bash
$ npm install is-nine
```

## Usage

``` js
var nine = require('is-nine');

if(nine(9))
{
  console.log("It is nine indeed!");
}
else
{
  console.log("It is not nine. What a shame!");
}
```
